Magento 2 Order Attachment

Magento 2 Order Attachment helps manage order data and add attachment along with the corresponding order.

Online stores have the limitation to explain the products to customers as it happens in the physical store. However, it can be overcome in Magento 2 store using the Magento 2 Order Attachment extension! It enables attaching docs such as screenshots, manuals, tutorials, designs, invoices, etc. with the orders.

The extension allows customers to attach files with the orders that can be viewed by customers and admin. It eases the task of retrieving the order data.

The Magento 2 Order Attachment extension allows communication with potential customers as it offers a feature to add comments with the attachments by admin and customers. 

Benefits of Choosing Magento 2 Order Attachment:

  • Let customers attach order documents and files at checkout, order review and My Account Order View page from the frontend.
  • Admin has the option to attach order docs from the Order View section in Backend.
  • Admin can limit the file size and restrict the file types that are uploaded by customers from the frontend.
  • Customers can delete the attachment files that are uploaded by them.
  • Admin can configure the module to receive notifications when customers attach or modify the attachment files. Specify the email ID in the backend to receive these notifications.
  • Admin and customers can add comments while uploading order attachments.
  • Attach multiple files along with a single order.
  • Admin is allowed to edit or remove attachment or set it as hidden from the backend.
  • Restrict attachments based on customer groups.
  • Let customers see their attachments in "My Order Attachments" tab in their "My Account" section.
